# 用AI打造你的第二收入曲线｜Landing Page

这是靖祥 Charlie 免费报告的 GitHub Pages 落地页。

## 页面

- `index.html`：报告介绍与 GetResponse 订阅表格
- `thank-you.html`：提交后的邮箱提醒页面
- `privacy.html`：简明隐私说明

## 发布

在仓库 `Settings → Pages` 中选择：

- Source：Deploy from a branch
- Branch：main
- Folder：/ (root)

## GetResponse 设置

1. Web Connect 和嵌入式表单代码已经加入 `index.html`。
2. 在 GetResponse 表单设置中加入页面网址：`https://jingxiangcharlie.github.io/ai-income-report/`
3. 把提交成功后的跳转网址设置为：`https://jingxiangcharlie.github.io/ai-income-report/thank-you.html`
4. 自动回复邮件中的报告按钮链接：`https://drive.google.com/file/d/1Q-NWddsSijjSGkjY_VB5DcjXQ5NXlzTp/view?usp=sharing`
